Lista kroków

1.Wybór pivota:

1.1. Wybierz pivot (element, który będzie używany do podziału tablicy).
1.2. Najczęściej wybiera się element na końcu tablicy, ale można używać innych metod wyboru (np. środkowy element, losowy element).

2.Podział tablicy:

2.1. Rozpocznij od porównania elementów tablicy z pivotem.
2.2. Przesuwaj wskaźniki z lewej i prawej strony tablicy:
- Lewy wskaźnik: Przesuwaj w prawo, dopóki nie znajdziesz elementu większego od pivota.
- Prawy wskaźnik: Przesuwaj w lewo, dopóki nie znajdziesz elementu mniejszego od pivota.
2.3. Jeśli oba wskaźniki nie przekroczyły się, zamień te dwa elementy miejscami.

3. Zamiana pivota:

3.1. Po zakończeniu przesuwania wskaźników, zamień pivot z elementem na pozycji, do której dotarły wskaźniki. Pivot będzie teraz na właściwej pozycji, a wszystkie elementy po lewej stronie są mniejsze, a po prawej większe.

4. Rekurencyjne sortowanie podtablic:

4.1. Teraz masz dwie podtablice: jedną zawierającą elementy mniejsze niż pivot, drugą zawierającą elementy większe niż pivot.
4.2. Rekurencyjnie wykonaj te same kroki dla obu podtablic (mniejszej i większej) aż do momentu, gdy tablica będzie już posortowana.
4.3. Przypadek bazowy: Jeśli tablica ma jeden element lub jest pusta, jest już posortowana.

5. Koniec algorytmu:

5.1. Algorytm kończy się, gdy wszystkie podtablice zostaną posortowane i cały zbiór danych będzie uporządkowany. 


                
Przykładowy kod:


                public class Sortowanie {
                    public static void szybkieSort(int[] arr, int low, int high) {
                        if (low < high) {
                            int pi = partition(arr, low, high);
                
                            quickSort(arr, low, pi - 1);
                
                            quickSort(arr, pi + 1, high);
                        }
                    }
                
                    public static int partition(int[] arr, int low, int high) {
                        int pivot = arr[high];
                        int i = (low - 1);
                
                        for (int j = low; j < high; j++) {
                            if (arr[j] <= pivot) {
                                i++;
                                int temp = arr[i];
                                arr[i] = arr[j];
                                arr[j] = temp;
                            }
                        }
                
                        int temp = arr[i + 1];
                        arr[i + 1] = arr[high];
                        arr[high] = temp;
                
                        return i + 1;
                    }
                
                    public static void printArray(int[] arr) {
                        for (int i = 0; i < arr.length; i++) {
                            System.out.print(arr[i] + " ");
                        }
                        System.out.println();
                    }
                
                    public static void main(String[] args) {
                        int[] arr = {64, 25, 12, 22, 11};
                
                        System.out.println("Tablica przed posortowaniem:");
                        printArray(arr);
                
                        szybkieSort(arr, 0, arr.length - 1);
                
                        System.out.println("Tablica po posortowaniu:");
                        printArray(arr);
                    }
                }
