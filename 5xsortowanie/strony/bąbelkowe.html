Lista kroków

1. Rozpocznij iterację po tablicy:

1.1. Zewnętrzna pętla iteruje przez wszystkie elementy tablicy.

2. Porównaj sąsiednie elementy:

2.1. Dla każdego elementu porównaj go z następnym elementem (elementami w parze).

3. Wymień miejscami, jeśli w złej kolejności:

3.1. Jeśli bieżący element jest większy niż następny, zamień je miejscami.

4. Powtarzaj krok 2 i 3:

4.1. Dla każdej pary elementów wykonuj porównanie i zamianę, przesuwając największe elementy na koniec tablicy w każdej iteracji.

5. Zmniejsz zakres pętli wewnętrznej:

5.1. Po każdej pełnej iteracji zewnętrznej pętli, elementy na końcu tablicy są już posortowane, więc zmniejszamy zakres wewnętrznej pętli.

6. Zakończenie:

6.1. Algorytm kończy się, gdy tablica jest w pełni posortowana.


                
Przykładowy kod:


                public class Sortowanie {
                    public static void babelkoweSort(int[] arr) {
                        int n = arr.length;
                        
                        for (int i = 0; i < n - 1; i++) {
                            for (int j = 0; j < n - 1 - i; j++) {
                                if (arr[j] > arr[j + 1]) {
                                    int temp = arr[j];
                                    arr[j] = arr[j + 1];
                                    arr[j + 1] = temp;
                                }
                            }
                        }
                    }
                
                    public static void printArray(int[] arr) {
                        for (int i = 0; i < arr.length; i++) {
                            System.out.print(arr[i] + " ");
                        }
                        System.out.println();
                    }
                
                    public static void main(String[] args) {
                        int[] arr = {64, 34, 25, 12, 22, 11, 90};
                
                        System.out.println("Tablica przed posortowaniem:");
                        printArray(arr);
                
                        babelkoweSort(arr);
                
                        System.out.println("Tablica po posortowaniu:");
                        printArray(arr);
                    }
                }
