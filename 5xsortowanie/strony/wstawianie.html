Lista kroków

1. Zaczynamy od drugiego elementu (indeks 1) w tablicy, ponieważ pierwszy element jest już uważany za posortowany.

2. Wybieramy bieżący element (klucz), który będziemy wstawiać w odpowiednie miejsce w posortowanej części tablicy.

3. Przesuwamy elementy, które są większe od klucza, o jedno miejsce w prawo. Przesuwamy elementy, aż znajdziemy odpowiednią pozycję dla klucza.

4. Wstawiamy klucz w znalezione miejsce.

5. Powtarzamy kroki 2-4 dla każdego kolejnego elementu w tablicy, aż cała tablica będzie posortowana. 


                
Przykładowy kod:


                public class Sortowanie {
                    public static void sortowanieWstaw(int[] array) {
                        for (int i = 1; i < array.length; i++) {
                            int key = array[i];
                            int j = i - 1;
                
                            while (j >= 0 && array[j] > key) {
                                array[j + 1] = array[j];
                                j--;
                            }
                            array[j + 1] = key;
                        }
                    }
                
                    public static void main(String[] args) {
                        int[] array = {5, 2, 9, 1, 5, 6};
                
                        System.out.println("Przed sortowaniem:");
                        for (int num : array) {
                            System.out.print(num + " ");
                        }
                
                        sortowanieWstaw(array);
                
                        System.out.println("\Po sortowaniu:");
                        for (int num : array) {
                            System.out.print(num + " ");
                        }
                    }
                }
